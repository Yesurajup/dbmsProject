RELATIONAL ALGEBRA

Select

The select operation is used for condition

Syntax

σ (Cond)(Relation Name)

Query

σ age>20(donor)

Projection

Projection operator is used to project particular columns from a relation

Syntax

∏(Column 1,Column 2….Column n)(Relation Name)

Query

∏dname(donor)

Rename

Rename operator is used to give another name to a relation

Syntax

ρ(Relation2, Relation1)

Query

ρ(donor1,donor)
